### Dependencies
#### Windows
* Qt Creator + Qt(4.7+)
* msvc2010
* python
* git


#### Linux
* Qt Creator + Qt(4.7+ | Qt5)
* python
* git

### Build
* check out

```
git checkout git@github.com:zhangshine/MdCharm.git
```
* Open "MdCharm.pro" by Qt Creator and click "Build"
