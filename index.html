Features

Live Preview

MdCharm will convert markdown into html and show the result in the preview panel when you make some changes.

Export

Export To HTML/PDF/ODT

Open One Directory as a Project

You can quickly open markdown files by double click it in the Project panel.

Custom CSS

Use your own CSS in the preview panel. Custom css in Settings->Preference->Styles

Show Line Numbers

By default, this feature is disabled. Enable it in Settings->Preference panel

Highlight Current Line

Find the caret easily(This feature is disabled by default, enable it in Settings->Preference panel).

Tabbed Documents

Open multiple markdown files in a single MdCharm application.

Find and Replace

Find and replace text

Drag and Drop

Drag markdown files and then drop them to MdCharm, let MdCharm to open all of them.
