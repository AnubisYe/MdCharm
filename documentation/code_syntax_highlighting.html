Custom Your Own Code Syntax Highlighting Style

If you have custom your own css before V0.9.8, MdCharm will append some code(below) to your custom style sheet file.
To create a custom style see the class reference in http://github.com/isagalaev/highlight.js/blob/master/classref.txt
